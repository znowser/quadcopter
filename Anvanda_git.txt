=== sätt upp git ===
1. Bli inbjuden av martin.
2. ladda hem git från http://git-scm.com/
3. Installera den nedladdade filen.
4. Ställ dig i förslagsvis mappen mina dokument, högerklicka och välj "git Bash".
5. gits konsollfönster öppnas nu. Skriv  git clone git@github.com:znowser/quadcopter.git
Git frågar eventuellt efter ditt gitkontos lösenord, skriv in det. Klart!



=== Lägg till/uppdatera fil ===
1. git add "filnamn"
filen "filnamn" är nu staged.
Observera att tomma mappar inte går att lägga till.
Alla filer som är staged kan visas med kommandot: git status

2. git commit -m "meddelande"
Filen "filnamn" är nu uppdaterad på din lokala kopia av repositoryn.
Observera att ett meddelande alltid måste finnas 

3. git push
git frågar efter ditt git-lösenord, skriv in det.
din fil är nu uppdaterad på den globala repositoryn.

================================



=== Ta bort fil ===
1. git rm "filnamn"
Detta kommando registrerar att filen "filnamn" markerats för borttagning.
git -r "mapp" används för att ta bort hela mappar.
Filer som markerats för borttagning kan visas med kommandot git status.

2. git commit -m "meddelande"

3. git push
Skriv in lösenord. Filen är nu borttagen.

================================



=== Ladda hem senaste ändringar ===
Detta skall göras varje gång INNAN du börjar ändra något i koden. Jobbiga
conflicts kan uppstå annars.

1. git pull
2. skriv in gitlösenord. klart!

================================
